SetLayerd

ウィンドウの半透明機能をON/OFFします。

[VB6]
Public Function SetLayerd( _
    ByVal hWnd As Long, _
    ByVal bLayerd As Boolean _
) As Long

[C++]
int SetLayerd(
    HWND hWnd, 
    bool bLayerd
);

パラメータ

hWnd
このパラメータで指定したウィンドウの半透明機能をON/OFFします。
bLayerd
trueを指定した場合、半透明機能をON。falseを指定した場合、半透明機能をOFFします。

戻り値

正常終了のときは0を、設定できなかったときは-1を返します。エラーが発生した場合は、Win32のエラーコードを戻します。

説明

hWndで指定したウィンドウの半透明機能をON/OFFします。 半透明機能をONにすることができないウィンドウをONにしようとすると、失敗します。

半透明機能をONにした場合、直後にSetOpacity関数を使って不透明度を設定する必要があります。

使い方は以下のとおり：

[VB6]
Dim ret As Long
ret = SetLayerd(Me.hWnd, True)
If ret = 0 Then
    SetOpacity Me.hWnd, 0.5
Else
    MsgBox "半透明機能をONにできません。"
Else

[C++]
int ret = SetLayerd(hWnd, true);
if (ret == 0) {
    SetOpacity(hWnd, 0.5);
} else {
    printf("半透明機能をONにできません。\n");
}
